# Análisis de Genomas Bacterianos
Proyecto bioinformático para el ensamblaje híbrido y análisis de cepas bacterianas usando datos de secuenciación provenientes de las tecnologías de Nanopore e Illumina.

## Objetivos del proyecto

- Realizar control de calidad de datos de secuenciación provenientes de las tecnologías de Nanopore e Illumina.
- Ejecutar ensamblaje híbrido de genomas bacterianos.
- Anotar funcionalmente los genomas ensamblados.
- Visualizar resultados
